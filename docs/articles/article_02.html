Now for the code to Flattening!

let arrays = [[1,2,3], [4,5], [6]];
console.log(arrays.reduce((flat, current) => flat.concat(current), []));

Step 1

This is involving an array.

Step 2

We are flattening a nested array by using the reduce method. We declare the variable arrays and assign it the numbers you see nested that we are trying to flatten.

Step 3

We use the console.log to print out the current array with the flattened one, by using the concat method to merge the two. We have an empty array which will be used for to store the flatted array in.

Step 4

The reduce method iterates for us each element in the arrays variable and applies the reducer function to then return the flattened array for us. As stated before the console.log statment is for the output of the flattened array to be shown to us.

IN CONCLUSION

This was to show by using javascript language how to flatten nested arrays by using the reduce method.
